PROJECT OVERVIEW
================================================================================
Healify is a comprehensive web-based healthcare platform that connects patients
with qualified medical professionals through online consultations. The platform
features a modern, responsive design with glassmorphism effects and a purple
gradient theme (#667eea to #764ba2).

TECHNOLOGY STACK
================================================================================
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Design: Glassmorphism, responsive design, mobile-first approach
- Icons: Font Awesome 6.4.0
- Storage: LocalStorage API for client-side data persistence
- No backend framework (ready for API integration)

JAVASCRIPT MODULES
================================================================================

1. UI.JS (User Interface Controller)
--------------------------------------------------------------------------------
Purpose: Main UI functionality and component loading

Functions:
- loadComponent(elementId, componentPath)
  * Fetches HTML component files
  * Injects into placeholder elements
  * Cache-busting with timestamp
  * Re-initializes event listeners after load
  * Error handling with fallback

- setupEventListeners()
  * Search bar toggle and functionality
  * Theme toggle (dark/light mode)
  * Profile button click handler
  * Sidebar toggle
  * Scroll effects

- checkAuthStatus()
  * Checks localStorage for user token
  * Updates UI based on login status
  * Shows/hides auth-required elements

- showNotification(message, type)
  * Displays toast notifications
  * Types: success, error, warning, info
  * Auto-dismiss after 3 seconds

- toggleTheme()
  * Switches between light and dark mode
  * Saves preference to localStorage
  * Updates theme icon

- toggleSidebar()
  * Opens/closes sidebar navigation
  * Toggles overlay
  * Prevents body scroll when open

- handleSearch(query)
  * Performs search across platform
  * Returns relevant results
  * Highlights matching text

- debounce(func, wait)
  * Utility function for performance
  * Delays execution until pause in calls

- smoothScrollTo(element)
  * Smooth scroll animation
  * Used for navigation links

RESPONSIVE DESIGN
================================================================================

Mobile-First Approach:
- Base styles designed for mobile (320px+)
- Progressive enhancement for larger screens
- Touch-friendly buttons (min 44x44px)
- Optimized images for mobile

Breakpoint Strategy:
- 480px: Stack elements, increase padding
- 768px: Two-column layouts, show/hide elements
- 1024px: Three-column layouts, full sidebar
- 1200px+: Maximum width containers, enhanced spacing

Mobile Optimizations:
- Hamburger menu for navigation
- Collapsible sections
- Simplified forms
- Reduced motion for animations
- Optimized font sizes
